# energy-consumption-forecasting
# Energy Consumption Forecasting with Pattern Discovery

Dataset:
Household Power Consumption Dataset

Objective:
Analyze energy consumption patterns and forecast future demand using machine learning.

Techniques Used:
- Data preprocessing using Pandas
- Feature extraction (daily average, variance, peak hours)
- K-Means clustering
- Random Forest Regressor
- Gradient Boosting Regressor

Evaluation Metrics:
- RMSE
- MAE
- R2 Score

Conclusion:
Cluster-based forecasting improves prediction accuracy by grouping households with similar energy usage patterns.
